# proyecto movil de reservas de hotel
----------
## Descripción
Este proyecto es una aplicación móvil desarrollada con kotlin para gestionar reservas de hotel.
permite a los usuarios buscar hoteles, ver detalles, hacer reservas y gestionar sus reservas existentes.todo lo anterior de manera local para el aprendizaje del manejo de base de datos sqlite en android.

## características principales
- Busqueda de hoteles
- Detalles del hotel
- Hacer reservas
- Gestionar reservas
- Autenticación de usuarios

## Tecnologías utilizadas
- Kotlin
- Android SDK
- sqlite
- XML para diseño de interfaces
- Git para control de versiones

## Instalación
1. Clona el repositorio: `git clone 
2. Abre el proyecto en Android Studio.
3. construye y ejecuta la aplicación en un emulador o dispositivo físico.
